# AI Hotspot Tracker - 需求文档

## 项目概述

**项目名称**：AI Hotspot Tracker（AI热点追踪器）
**目标用户**：AI编程博主
**核心诉求**：第一时间自动发现AI领域热点（如大模型更新），及时通知用户，走在"吃瓜第一线"

---

## 功能需求

### F1：关键词监控

**用户故事**：作为一名AI博主，我想输入关键词（如"GPT-5"、"Claude 4"），系统自动监控这些关键词相关的内容出现，并第一时间通知我。

**详细需求**：
- 用户可以添加、删除、暂停/恢复关键词
- 系统从多个信息源实时搜索关键词相关内容
- AI 识别内容的真伪和可信度，过滤虚假/炒作内容
- 当高可信度内容与关键词匹配时，立即发送通知
- 通知包含：标题、AI摘要、来源、可信度、相关度评分
- 用户可设置最低相关度阈值（默认0.7）
- 用户可选择仅通知高可信度内容

**验收标准**：
- [ ] 添加关键词"GPT-5"后，系统在30分钟内发现相关新内容并通知
- [ ] 低可信度/虚假内容不会触发通知（除非用户主动查看）
- [ ] 通知到达延迟 < 1分钟（从发现到推送）

### F2：热点自动发现

**用户故事**：作为一名AI博主，我想系统每隔一段时间自动搜集"AI编程"范围内的热点，让我不用手动搜索也能了解最新动态。

**详细需求**：
- 用户可设置监控范围（如"AI编程"、"大模型"、"AI工具"）
- 系统定期从多个数据源聚合热点内容
- AI 自动分析：摘要生成、热度评分、可信度评估、关键词分类
- 热点按热度/时间排序展示
- 支持按来源、可信度、关键词筛选

**验收标准**：
- [ ] 系统每15-30分钟自动采集一次
- [ ] 5个数据源（Twitter、网页搜索、HN、GitHub、RSS）同时工作
- [ ] AI自动为每条内容生成摘要和评分
- [ ] 热点列表支持排序和筛选

### F3：实时通知

**用户故事**：当重要热点出现时，我要立即收到通知，不管我是否在看网页。

**详细需求**：
- 浏览器桌面推送通知（需用户授权）
- 站内通知中心（带已读/未读标记）
- 通知类型：关键词匹配、高热度爆发、可信度警报
- 支持通知静音（暂停N小时）

**验收标准**：
- [ ] 关键词匹配时触发浏览器桌面通知
- [ ] 站内通知列表展示所有历史通知
- [ ] 新通知实时出现在通知中心，无需刷新

### F4：信息真伪识别

**用户故事**：作为一名负责任的博主，我需要AI帮我判断信息的可信度，避免传播虚假消息。

**详细需求**：
- AI 基于多维度评估可信度：来源可靠性、内容证据、与已知信息一致性
- 可信度分级：高（绿色）、中（蓝色）、低（琥珀色）、虚假（红色）
- 低可信度内容有明显警告标记
- 可查看AI的判断依据

**验收标准**：
- [ ] 每条内容都有可信度评估和评分
- [ ] 低可信度内容有醒目警告
- [ ] 可查看AI判断依据

### F5：Agent Skills 封装（第二阶段）

**用户故事**：我想把监控能力封装成技能，让其他AI（如Claude Desktop）也能调用。

**详细需求**：
- 封装为 MCP Server
- 提供工具：track_keywords、get_trending、check_credibility、get_keyword_matches
- 可在 Claude Desktop / Cursor 等支持 MCP 的客户端中使用

**验收标准**：
- [ ] MCP Server 可独立运行
- [ ] 4个工具均可正常调用并返回数据
- [ ] Claude Desktop 中配置后可使用

---

## 非功能需求

### N1：产品形态
- 响应式Web页面，兼容桌面端和移动端
- 前端设计独特，采用"信号雷达"主题风格

### N2：性能
- 页面加载 < 3秒
- SSE推送延迟 < 1秒
- 支持100+关键词同时监控

### N3：数据源多样性
- 至少5个独立数据源
- 单一数据源不可用时不影响整体功能

### N4：成本控制
- AI调用日成本 < $0.50
- 尽量利用免费数据源
- 避免重复调用AI分析同一内容

## 数据源

| 数据源 | 类型 | 接入方式 | 频率 |
|--------|------|---------|------|
| Twitter/X | 社交媒体 | twitterapi.io API | 30min |
| 网页搜索 | 综合新闻 | 搜索引擎爬虫 | 30min |
| HackerNews | 技术社区 | 官方API | 15min |
| GitHub Trending | 开源项目 | HTML爬取 | 60min |
| RSS订阅 | 博客/媒体 | rss-parser | 20min |
